Homework Two: ShuffleMusicPlayer Class And Object Inheritance

The second homework in this class had us working with some java graphics, which I was surprised to find were very simple. It also deals with some basic object inheritance from the Playlist class to the ShufflePlaylist class. The overall goal of the homework was to make a small little graphics application, though the only thing we had to do was extend the PLaylis object to enable the Shuffle feature of the application.

The files for the project can be found here.

My opinion: This lesson was actually somewhat interesting! I'm glad we've moved onto some graphics work, and some more complicated object-related topics like inheritance. Hopefully soon we will be moving on to even more complicated topics, as I was able to do this pretty quickly.
